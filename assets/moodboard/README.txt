PRESS IT — moodboard slots
==========================

Drop image files in this folder and they appear in the PRESS IT section
automatically. No config, no rebuild, no code to touch.

Name them in order:

    01.jpg
    02.jpg
    03.jpg
    ...  up to  20.jpg

.jpg, .jpeg, .png, .webp and .avif all work.

Number them consecutively with no gaps. The page walks the numbers in order
and stops at the first one it can't find, so if you have 01, 02 and 04, only
the first two show up.

If the folder is empty, the moodboard section stays hidden and the CSS-native
Press It layout stands on its own. Nothing breaks either way.

The site renders them in greyscale and returns them to full colour on hover,
which is the same treatment the discography grid uses.

A note on what you put here
---------------------------
The album cover already on the page comes from the Apple Music catalogue and
is fine to hotlink. Anything you add to this folder is your own curation — if
you plan to make the site public, favour images you have the right to use, or
credit the photographers. Fansite-taken photos usually carry a "no edit / no
reupload" rule from the person who shot them.
